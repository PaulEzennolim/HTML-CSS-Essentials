Accessibility

Our commitment to an inclusive web experience

Accessibility Statement

This website aims to conform to WCAG 2.1 Level AA. We are committed to ensuring all users can access our content regardless of ability or technology.

Our Accessibility Measures

Semantic HTML5: We use elements such as <header>, <nav>, <main>, <section>, and <footer> so screen readers can identify page regions correctly, as recommended by WAI page structure guidelines.

Colour contrast: Text and background colours meet a minimum contrast ratio of 4.5:1, verified with the WebAIM Contrast Checker.

Alternative text: All images include descriptive alt attributes. Decorative elements use aria-hidden="true".

Keyboard navigation: A skip-to-content link and visible focus styles ensure full keyboard operability.

Form labels: Every form input is associated with a <label> element and uses aria-describedby for additional hints.

Assistive Technology

This site is designed to work with screen readers such as NVDA (free, Windows), VoiceOver (built-in, macOS/iOS), and TalkBack (Android). Browser zoom up to 200% is fully supported without loss of content.
